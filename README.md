# Teknikkk PK Sprint

Static Penalaran Kuantitatif sprint practice site.

## Local Run

```powershell
cd study_web
python -m http.server 4173
```

Open `http://127.0.0.1:4173/index.html`.

## GitHub Pages

This repo uses GitHub Actions to publish the `study_web` folder to GitHub Pages.
After pushing to `main`, enable Pages with source `GitHub Actions` in repository settings if it is not already enabled.
